UQOMM Firmware Programming Guide

Professional Installation & Configuration Manual
1 Requirements & Prerequisites

Ensure you have the following components before starting:

  • STM32CubeProgrammer Software (latest version)
  • ST-LINK/V2 Programmer or compatible interface
  • An available USB A port for programmer connection (Use a USB Hub if needed)
  • Target STM32 Device to be programmed
  • Firmware File (.hex, .bin, or .elf format)
  • SWD/JTAG connection cables
  • Proper power supply (3.3V or 5V)
2 STM32CubeProgrammer Installation

Download and install STM32CubeProgrammer from STMicroelectronics. The package includes all necessary drivers for programming UQOMM devices.

System Requirements: Windows 10/11 (64-bit), 2GB RAM, 500MB disk space, USB 2.0 port.

3 Hardware Setup - Programmer Connection

Connect your ST-LINK programmer to the target UQOMM device. Proper hardware connection is critical for successful programming.

âš  Critical Requirement

A dedicated programmer is mandatory. Ensure proper SWDIO, SWCLK, GND, and VCC pin connections. Incorrect wiring may damage hardware.

SWD Pinout: SWDIO (Data), SWCLK (Clock), GND (Ground), VCC/VDD (3.3V), NRST (Reset - optional)

4 Launch STM32CubeProgrammer

Open STM32CubeProgrammer on your workstation. Wait for the interface to fully load before proceeding.

5 Connect to Target Device

Before flashing new firmware, the device should be disconnected from power. After that, plug the ST-Link and click "Connect" to establish communication. The software will detect the device and display relevant information (part number, flash size, firmware status).

6 Load Firmware File

In STM32CubeProgrammer, click "Open File" and select the firmware file (.hex, .bin, or .elf). The SKU of the target product is specified in its file name, so make sure to verify it before flashing.

Click "Download" to begin firmware transfer. The software will erase flash sectors, program new firmware, verify data, and display completion status.
